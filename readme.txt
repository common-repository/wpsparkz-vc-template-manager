VC Template Manager is a flexible template manager to manage your templates. Whether you're a beginner or a professional, you're going to love taking control of your templates. == Installation ==
1. First install Visual Composer Plugin. <a href="https://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=wpsmsparkz">Here</a> is link.

2. Download the latest zip file and extract the `wpsparkz-vc-templates` directory.

3. Upload it to your `/wp-content/plugins/` directory.

4. Activate `VC Template Manager` on the `Plugins` menu in WordPress.
